# SOC L1 Alert Reporting

Learn how to properly report, escalate, and communicate about high-risk SOC alerts.

## Focus

- Capture the key evidence from an alert investigation.
- Summarise the risk clearly for handoff or escalation.
- Document impact, scope, and recommended next steps.
- Keep reporting concise, accurate, and actionable.
